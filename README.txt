The Splunk Add-on for Amazon Web Services allows a Splunk Enterprise administrator to collect events, alerts, performance metrics, and billing information from the CloudWatch and CloudTrail services. It can also gather log data from generic S3 buckets. This add-on provides modular inputs and CIM-compatible knowledge to use with other Splunk Enterprise apps, such as Splunk App for Enterprise Security and Splunk App for PCI Compliance.
